Was wird benötigt (Installationsbefehle für Ubuntu):
midori - Browser: sudo apt-get install midori
NodeJs : sudo apt-get install nodejs
NPM : sudo apt-get install npm
dnsmasq : sudo apt-get install dnsmasq
NowJs : npm install now
